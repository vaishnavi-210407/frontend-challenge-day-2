🚀 Day 2 / 21 — Theme Toggle (Dark / Light Mode)
📌 Project Name

Theme Toggle

🎯 Goal

User easily dark mode / light mode switch kar sake aur uski preference page reload ke baad bhi save rahe.

🧩 Features
🌞 Light Mode
🌙 Dark Mode
💾 localStorage me theme save
🔄 Page reload ke baad same theme apply
⚡ Smooth UI transition
🔄 Step-by-Step Flow
User toggle button par click karega
JavaScript theme change karega (light ↔ dark)
Current theme localStorage me store hogi
Page reload hone par saved theme check hoga
Automatically wahi theme apply hogi
🖥 UI Plan
Toggle button / switch
Main container
Heading / title
Background gradient (light & dark)
⚙️ Logic Plan
Button par click event
classList.toggle() use karke theme change
localStorage.setItem() se theme save
window.onload par saved theme apply
🚧 Edge Cases
Page first time open → default theme set
localStorage empty ho
Rapid clicking (toggle spam)
Browser storage disabled
🛠 Tech Stack
HTML
CSS
JavaScript
